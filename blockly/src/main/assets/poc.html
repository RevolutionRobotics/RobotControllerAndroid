Robotics POC

This HTML is the Proof of Concept for Javascript bridging. We need to validate that Javascript can call native methods with parameters, as well as handling their return value.

To validate this, the link below calls two native methods of the NativePoc class: createMessage(String): String and displayMessage(String).

  • createMessage(String): String should return the parameter string with ' World' appended to it. For example, if the parameter is 'Hello', the returned string should be 'Hello World'.
  • displayMessage(String) should show a Toast/Alert with the message received as a parameter.
